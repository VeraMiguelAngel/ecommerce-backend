# 🛒 Ecommerce Backend

Proyecto integrador individual del **Módulo 4 - Bootcamp Henry**, especialización en **Backend**.  
Desarrollo del backend para una aplicación de ecommerce, utilizando **NestJS, PostgreSQL, TypeORM y TypeScript**.  

---

## 👨‍💻 Mi rol en el proyecto
- Desarrollo completo del **backend**.  
- Configuración de base de datos relacional con **PostgreSQL** y **TypeORM**.  
- Implementación de autenticación y autorización con **JWT**.  
- Integración de subida de imágenes con **Cloudinary**.  
- Documentación de la API con **Swagger**.  

---

## 🚀 Tecnologías utilizadas
- **Backend:** NestJS, TypeScript, Node.js  
- **ORM:** TypeORM  
- **Base de datos:** PostgreSQL  
- **Autenticación:** JWT  
- **Storage:** Cloudinary  
- **Documentación:** Swagger  

---

## 📦 Funcionalidades principales
- CRUD de productos, usuarios y órdenes.  
- Autenticación y autorización con JWT.  
- Gestión de imágenes de productos con Cloudinary.  
- Endpoints documentados con Swagger UI.  

---

## ⚙️ Instalación y uso
1. Clonar el repositorio:  
   ```bash
   git clone https://github.com/VeraMiguelAngel/ecommerce-backend.git
2. Instalar dependencias:
   
  npm install

3. Configurar variables de entorno (.env):

DB_HOST, DB_USER, DB_PASS, DB_NAME

CLOUDINARY_URL

JWT_SECRET

4. Ejecutar en modo desarrollo:
  npm run start:dev

5. Acceder a la documentación Swagger:
   ```bash
   http://localhost:3000/api
